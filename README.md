# 🚢 Titanic Dataset: Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs a comprehensive **Exploratory Data Analysis (EDA)** on the Titanic dataset. The objective is to identify key factors that influenced passenger survival using data analysis and visualization techniques.

---

## 🎯 Objectives

* Understand the structure and quality of the dataset
* Handle missing values effectively
* Explore relationships between features
* Identify patterns affecting survival rates

---

## ⚙️ Tools & Technologies

* **Python**
* **Pandas** – Data manipulation and preprocessing
* **Matplotlib & Seaborn** – Data visualization

---

## 🔄 Data Processing Steps

### 1. Data Cleaning

* Filled missing values:

  * `Age` → Median value
  * `Embarked` → Mode value
* Dropped `Cabin` column due to excessive missing values
* Converted key columns (`Survived`, `Pclass`, `Sex`, `Embarked`) into categorical data types

---

### 2. Exploratory Data Analysis

* Analyzed survival distribution across different features
* Grouped data by gender, passenger class, and age
* Generated summary statistics for insights

---

## 📊 Key Insights

* 👩 Female passengers had a significantly higher survival rate than males
* 🏆 First-class passengers had the highest survival probability
* 🔍 Third-class passengers had the lowest survival rate
* 📈 Strong relationship between **gender, class, and survival**

---

## 📉 Visualizations

* Survival count plots
* Heatmaps showing feature relationships
  n- Categorical plots for class and gender analysis

## ▶️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/Redietabay/Exploratory-Data-Analysis-on-Titanic-Dataset.git
   ```

2. Install required libraries:

   ```bash
   pip install pandas matplotlib seaborn
   ```

3. Place the dataset:

   * Add `train.csv` inside the project folder

4. Run the notebook:

   * Open `Task_2_EDA.ipynb` using Jupyter Notebook or VS Code
